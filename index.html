La marca

Italia, Atlántico y una identidad visual con carácter.

Esta versión del sitio toma la ola, la paleta y la gráfica principal del sistema de marca para construir una presencia online coherente desde el primer día.

Marvento se apoya en una marca gráfica de inspiración italiana de los años 60, con una tipografía robusta, presencia fuerte y un lenguaje visual que une el mar, el viento y el ritual del aperitivo.

En esta etapa usamos las piezas reales del sistema visual para que la web respire exactamente el mismo universo gráfico del proyecto de marca.

Productos

Dos expresiones. Una misma identidad.

La línea principal está compuesta por Marvento Rosso y Marvento Bianco, presentados con la gráfica, el color y la construcción visual del sistema original.

Rosso

Marvento Rosso

Vermut con más carácter, profundidad y presencia en copa. Ideal para un aperitivo clásico, con hielo y cítrico, o para cócteles de perfil más intenso.

Bianco

Marvento Bianco

Una expresión más fresca, aromática y amable. Pensada para un servicio liviano, con mucho hielo, soda o tónica, y una lectura más refrescante del aperitivo.
